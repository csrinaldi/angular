Como se compone?

  • Shadow DOM: Permite encapsular el DOM y CSS
  • HTML Templates: Es una forma de clonar el DOM para que este sea reutilizado en la pagina
  • Custom elements: Esto nos permite extender un elemento de HTML o bien crear uno nuevo.
  • HTML Imports: Es la manera de inclusión de otros WebComponent.

Shadow DOM

  • shadow root: Nuevo tipo de nodo
  • shadow host: Es todo nodo que tiene asociado un shadow root, El contenido del shadow host no es renderizado, en su lugar es renderizado el contenido del shadow root
                                
                                    
<button>Hello, world!</button>
<script>
    var host = document.querySelector('button');
    var root = host.createShadowRoot();
    root.textContent = 'こんにちは、影の世界!';
</script>

                                
                            
                                
<script>
    var host  = document.querySelector('button');
    var label = document.getElementById("content_shadow_host");
    label.value = host.textContent;
</script>

Slide Backgrounds

Set data-background="#007777" on a slide to change the full page background to the given color. All CSS color formats are supported.

Background Transitions

Pass reveal.js the backgroundTransition: 'slide' config argument to make backgrounds slide rather than fade.

You can override background transitions per slide by using data-background-transition="slide".
